# MacroLink — Macro Intelligence Terminal

Prototype autonome d'un terminal macro multi-actifs. Le site tient dans un seul fichier HTML et peut être déployé directement sur Vercel, Netlify ou n'importe quel hébergement statique.

## Fonctionnalités

- Radar de régime et facteurs macro
- Univers de 56 actifs avec recherche, filtres et sélection
- Matrice de corrélations factorielle et laboratoire de paires
- Huit scénarios macro avec intensité réglable
- Portefeuille long/short et stress tests
- Briefings structurés avec catalyseurs et invalidations

Les prix, variations, corrélations et analyses sont des données fictives de démonstration. Ils ne constituent pas des conseils financiers.
